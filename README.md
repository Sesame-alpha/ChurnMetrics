# ChurnMetrics
A telecoms customer churn risk dashboard. Single HTML file, no setup required.

What it does
1.Scores customers 0–100 based on tenure, usage decline, complaints, payment history, and plan type
2.Classifies customers as Low / Medium / High churn risk
3.Shows churn rate trend, risk score distribution, and package tier breakdown
4.Highlights revenue at risk from high-risk customers
5.Click a customer to filter the table
6.Filterable customer log with PDF export
"Generate new batch" reshuffles the data
